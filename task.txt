Реализуйте игру в крестики нолики с помощью класса GameField

Создайте класс GameField, со следующими обязательными свойствами и методами:

state - информация о клетках поля, изначально все клетки пустые;
mode - свойство для хранения информации активного игрока (крестики или нолики), первые ходят крестики;
isOverGame - игра закончена или нет;
getGameFieldStatus - метод для получения информации о состоянии игрового поля;
setMode - метод для изменения активного игрока;
FieldCellValue - метод для заполнения клетки поле (обязательно должна быть проверка занята ли клетка другим крестиком или ноликом) - вводите координаты столбец - строка с помощью модальных окон.set.
Вы можете написать свои свойства и методы, но в комментариях укажите информацию о работе.

Создайте объект класса и до тех пор пока кто-то не одержит победу скрипт должен выполняться. В конце игры выведите сообщение пользователю с результатом игры.

Пример:

class GameField {
    // тело класса
}

const gameField = new GameField();

while (!gameField.isOverGame) {
    //игра идёт
}

//вывод сообщения о результате игры
